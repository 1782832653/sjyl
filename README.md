# 使用说明
* 使用本程序前请先修改管理员密码 文件config.php
* 默认的管理员密码为admin 
* 管理员后台admin.php
* 本程序使用的是sqlite数据库 请确认你的服务器支持sqlite数据库
* 推荐使用adminer工具管理sqlite数据库，选择sqlite3 输入words.sqlite即可管理
* 如有其它任何问题请联系作者本人
